## Como utilizar

Para visualizar e editar os relatórios deste projeto, é necessário ter o **Microsoft Power BI Desktop** instalado.

### 1. Instalação do Power BI
Caso ainda não tenha o Power BI Desktop instalado, faça o download gratuitamente através do link oficial:
- [Download Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/)

### 2. Abrindo os arquivos
1. Após a instalação, navegue até as pastas deste repositório (ex: `LAB1`, `LAB2`).
2. Localize os arquivos com extensão `.pbix`.
3. Dê um duplo clique no arquivo desejado para abri-lo no Power BI Desktop.
